# 🤝 贡献指南

感谢你对 OpenClaw Console 的关注！

## 如何贡献

### 报告 Bug
1. 在 [Issues](https://github.com/xmgzxmgz/openclaw-console/issues) 页面创建新 Issue
2. 描述问题的复现步骤
3. 附上截图或日志

### 提交功能建议
1. 在 Issues 页面创建 Feature Request
2. 描述你想要的功能和使用场景

### 提交代码
1. Fork 本仓库
2. 创建分支：`git checkout -b feature/your-feature`
3. 提交：`git commit -m 'feat: add your feature'`
4. 推送：`git push origin feature/your-feature`
5. 创建 Pull Request

## 开发规范

- **零依赖**：保持原生 Node.js + 原生前端，不引入 npm 包
- **代码风格**：中文注释，简洁命名
- **提交信息**：使用 Conventional Commits 格式
  - `feat:` 新功能
  - `fix:` Bug 修复
  - `docs:` 文档更新
  - `style:` 样式调整
  - `refactor:` 代码重构
